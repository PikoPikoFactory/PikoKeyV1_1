# PikoKeyBeta

## About this page
- MIDIキーボード PikoKey Ver1.1版 組み立てキットの情報ページです。
- Pikoket 2 で頒布したβ版のバグフィックス版です。回路図に変更があります。
- 随時更新中

- Arduino Nano Everyは別売ですので、別途入手しておいてください。D20,D21を使用しているので、無印Nanoは使えません。
